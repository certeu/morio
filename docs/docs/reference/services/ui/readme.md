The `ui` service provides the web interface to Manage or interact with Morio.

<Note>
This service __is available__ in [ephemeral state](/docs/reference/terminology/ephemeral-state/)
</Note>

<Scode>
Source code for this service resides in [the `ui` folder of our monorepo](https://github.com/certeu/morio/tree/develop/ui).
</Scode>

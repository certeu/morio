---
title: Operator API
---

As an API-first project, Morio can be fully deployed, configured, and managed
through its Operator API. It is a REST API that facilitate operational
management of your Morio deployments.

It includes endpoints for initial setup and deployment of your Morio instances,
as well as endpoints to facilitate monitoring, observability, and devops
workflows.

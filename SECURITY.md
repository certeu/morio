# Security Policy

## Supported Versions

Morio is under active development, yet currently still in alpha.

| Version | Supported |
| ------- | --------- |
| < 1.0   | :x:       |

## Reporting a Vulnerability

To report a vulnerability, please contact CERT-EU via
[services@cert.europa.eu](mailto:service@cert.europa.eu).
